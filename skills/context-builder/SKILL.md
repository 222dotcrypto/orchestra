# Context Builder

Автоматическое построение контекста для задач на основе кодовой базы и документации.

## Триггеры
- `построй контекст`
- `собери контекст`
- `context build`

## Инструменты
- grep
- file reader

## Описание
Скилл анализирует кодовую базу и собирает релевантный контекст для выполнения задач.
Поддерживает разные форматы вывода.
