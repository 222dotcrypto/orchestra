## Оптимизации v2 и их эффект

### 1. Компактные промпты (50 → 10 строк)
- v1: каждый промпт содержал полный протокол (heartbeat, jq, правила)
- v2: протокол вынесен в WORKER_PROTOCOL.md, промпт = 10 строк
- **Эффект:** экономия ~40 строк × 5 воркеров = ~200 строк system prompt → меньше input-токенов

### 2. Сигнальные файлы вместо JSON polling
- v1: Мозг каждые 15 сек читал все task-*.json (3-6 Read calls × 35 циклов = ~150 tool calls)
- v2: Мозг делает `ls .brain/signals/` (1 call) + check-workers.sh (1 call) = 2 calls/цикл
- **Эффект:** ~150 → ~10 tool calls на мониторинг (2 цикла вместо 35)

### 3. Adaptive polling (3 мин пауза)
- v1: начинал мониторить сразу, каждые 15 сек
- v2: 3 минуты пауза, потом проверка. Воркеры обычно заканчивают за 3-5 мин
- **Эффект:** оба раза воркеры закончили за время паузы → 0 лишних циклов мониторинга

### 4. Строгие фазы
- v1: все 3 воркера запущены одновременно, тестер ждал кодера (~10 мин idle)
- v2: Phase 1 (3 coder) → ревью → Phase 2 (coder + tester). Никто не idle
- **Эффект:** 0 idle polling воркеров, 0 restarts

### 5. Interface-файл между фазами
- v1: не было, воркеры сами читали чужой код
- v2: phase-1-interface.md с сигнатурами и типами
- **Эффект:** воркеры Phase 2 точно знали интерфейсы → 0 rework

### 6. Примеры данных в description
- v1: "сделай парсер JSONL" (абстрактно)
- v2: конкретный пример строки JSONL, точные цены, пример таблицы
- **Эффект:** 0 rework на всех 5 задачах (vs 1 restart в v1)

## Что не изменилось
- Мозг по-прежнему дорогой (~$28 за сессию из-за большого system prompt CLAUDE.md)
- cache_read_input_tokens доминирует в стоимости (кэш контекста)
- Каждый воркер тратит ~$3-5 независимо от сложности задачи (baseline cost = загрузка контекста)

## Выводы
1. **Стоимость/задача упала с $17 до $5** — основной прорыв от сигнальных файлов и adaptive polling
2. **0 rework** — конкретика в description (примеры данных, формулы, ожидаемый output) = ключевой фактор
3. **Bottleneck = Мозг**: координация стоит столько же сколько вся работа воркеров
4. **Baseline cost воркера ~$2-3**: даже пустой воркер (загрузка контекста) стоит $2+. При <$2 работы выгоднее DIY
